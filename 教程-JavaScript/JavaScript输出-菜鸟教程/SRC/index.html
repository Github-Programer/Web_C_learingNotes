JavaScript输出有4种，至少现在是4种

        使用 window.alert() 弹出警告框。
        使用 document.write() 方法将内容写到 HTML 文档中。
        使用 innerHTML 写入到 HTML 元素。
        使用 console.log() 写入到浏览器的控制台。

JavaScript可以输出日期，调用Date()

JavaScript中的func()函数和function关键字先不用研究，知道是函数就行

但是有几个要注意的地方：

  • Date()：这个函数可以直接输出，调用日期
  • <button onclick = "func()">：button标签，可以在HTML中设置按钮，后面详解，onclick可以判定鼠标点击，
    还可以调用函数，在双引号中写出
